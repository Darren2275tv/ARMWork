STM32 Nucleo Board Firmware Application Package Update History

  

V1.0.0 / 11-February-2014

 

Main Changes

•  First official release.

    • Important Note: ST-LINK/V2-1 integrated in STM32 Nucleo Board is not yet supported by EWARM v6.70.3 and MDK-ARM v4.73.0 used for this package development,

     it will be supported from EWARM v7.10 and MDK-ARM v5.10.  Currently, user must install ST-LINK/V2-1 driver available on www.st.com to be able to use STM32 Nucleo board.

Content

• CortexM CMSIS V3.20 (Release Notes)

• STM32F0xx Standard Peripherals Driver V1.3.0 (Release Notes)

• CMSIS_Device_STM32F0xx V1.3.1 (release notes)

• STM32F10x Standard Peripherals Driver V3.6.1 (Release Notes)

• CMSIS_Device_STM32F10x V3.6.2 (release notes)

• STM32L1xx Standard Peripherals Driver V1.3.0 (Release Notes) 

• CMSIS_Device_STM32L1xx V1.3.0 (release notes)

• Demonstration & Peripheral firmware examples firmware V1.0.0 (release notes)

• STM32 Nucleo Drivers V1.0.0 (release notes)

 

Development Toolchains and Compilers 

• IAR Embedded Workbench for ARM (EWARM) toolchain v6.70.3

• A software patch is needed to use STM32L152RE devices with MDK-ARM v4.73.0. If you do not have

this patch installed, you can install it from the following folder "\Utilities\Third_Party\MDK-ARM_STM32L152RE_Patch

• RealView Microcontroller Development Kit (MDK-ARM) toolchain v4.73.0

 

Supported Devices and EVAL boards

•  STM32nucleo board Rev C. 

•  Devices supported : STM32FO30R8, STM32F103RB and STM32L152RE.
